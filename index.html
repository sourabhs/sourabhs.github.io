Welcome to the ZapApps Homepage

Following are two applications which I made as part of the Intern Application process for Zappos.com

  • WishList App
    This App was made as part of the challenge solution for the Java Development Intern Position.
    A user can input a Product ID or Product Name along with his/her email address and the app will send out an email to the given address anytime the item goes on a 20% or more discount.
    I used Java, HTML5, JSP, jQuery, Javascript along with JavaMail and JSON API(javax.json).
    Sample Inputs:
    ID - 7515478
    OR Name - tall
  • Gift Suggesting App
    This App was made as part of the challenge solution for the Software Engineering Intern Position.
    A user has to enter a desired amount which he/she wants to spend and the number of gifts he/she wants to buy. The App will make suggestions on which combinations of products are best suited as per the given inputs.
    I used ASP.NET, C#, HTML5 along with the JSON.Net package(by Newtonsoft).
    Sample Inputs:
    No of Items - 4
    Desired Amount - 600
